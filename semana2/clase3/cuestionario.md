1. El gráfico que permite visualizar valores extremos es:  
A) Histograma  
B) Barras  
C) Cajas  
D) Circular o torta  
**Respuesta: C**

2. El gráfico más adecuado para comparar categorías es:  
A) Barras  
B) Dispersión  
C) Histograma  
D) Cajas  
**Respuesta: A**

3. El gráfico que muestra la relación entre dos variables cuantitativas es:  
A) Circular o torta  
B) Dispersión  
C) Barras  
D) Cajas  
**Respuesta: B**

4. El histograma se utiliza para representar:  
A) Variables cualitativas  
B) Datos agrupados en intervalos  
C) Solo porcentajes  
D) Relaciones entre variables  
**Respuesta: B**

5. En una tabla de frecuencias, **fi** representa:  
A) Frecuencia acumulada  
B) Frecuencia relativa  
C) Frecuencia absoluta  
D) Marca de clase  
**Respuesta: C**

6. La fórmula de Sturges se usa para calcular:  
A) La media  
B) El número de clases  
C) La mediana  
D) La moda  
**Respuesta: B**

7. Una tabla de doble entrada permite estudiar:  
A) Una sola variable  
B) Dos variables  
C) Tres variables  
D) Solo porcentajes  
**Respuesta: B**

8. El gráfico circular o de torta se usa principalmente para mostrar:  
A) Relación entre variables  
B) Valores extremos  
C) Partes de un total  
D) Datos continuos agrupados  
**Respuesta: C**

9. La amplitud de clase se obtiene dividiendo:  
A) La media entre el rango  
B) El rango entre el número de clases  
C) El número de clases entre el rango  
D) La frecuencia entre el total de datos  
**Respuesta: B**

10. En una tabla bien elaborada, el título debe indicar qué, cómo, dónde y:  
A) Quién  
B) Cuánto  
C) Cuándo

11. El gráfico de cajas también recibe el nombre de:  
A) Histograma  
B) Caja y bigotes  
C) Dispersión  
D) Circular  
**Respuesta: B**

12. El número de clases depende principalmente de:  
A) La moda
